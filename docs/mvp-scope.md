# MVP Scope

## Inclus

- Recherche fédérée sur un nombre réduit de sources (3 à 5).
- Normalisation de résultats via un schéma commun.
- Galerie homogène de résultats.
- Ouverture simple et multi-manifest dans Mirador.
- Import manuel d’URL (manifest/notice) via backend.
- Endpoint `/api/sources` avec capacités des connecteurs.
- Robustesse aux succès partiels.

## Exclu

- Comptes utilisateurs.
- Annotation collaborative persistante.
- Moissonnage global/exhaustif du web patrimonial.
- Index mondial complet.
- OCR universel global.

## Critères de succès MVP

- Une requête simple renvoie des résultats depuis au moins 3 sources.
- Les résultats s’affichent dans une galerie unifiée.
- L’ouverture IIIF dans Mirador fonctionne pour la majorité des items manifestés.
- L’ouverture multi-objets fonctionne.
- L’échec d’une source n’interrompt pas la réponse globale.
